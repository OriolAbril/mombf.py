mombf: Bayesian Model Selection for Non-Local and Local Priors
==============================================================
 ``mombf.py`` is a JAX powered version of R package
 `mombf <https://github.com/davidrusi/mombf>`_. A package for Bayesian
 model selection in generalized linear models using non-local priors.
